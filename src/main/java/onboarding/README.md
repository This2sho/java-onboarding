## 문제 1

- 책을 임의로 펼친다.
  - 왼쪽 페이지는 홀수, 오른쪽 페이지는 짝수
  - 시작 면이나 마지막 면이 나오도록 책을 펼치지 않는다.
    - (1, 2) , (399, 400) 의 경우 return -1
- 점수를 구한다.(아래 중 가장 큰 수)
  - 왼쪽 또는 오른쪽 페이지 번호의 각 자리 숫자를 모두 더한다.
  - 왼쪽 또는 오른쪽 페이지 번호의 각 자리 숫자를 모두 곱한다.
- 점수를 비교해 가장 높은 사람이 게임의 승자.
